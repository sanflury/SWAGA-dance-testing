## Тест-кейс №1: Успешная отправка формы

**Описание:** Проверить, что форма успешно отправляется при заполнении всех обязательных полей корректными данными

**Предусловие:**
- Открыта главная страница сайта
- Модальное окно закрыто

**Шаги:**
1. Прокрутить страницу до блока "ВЫБЕРИ СВОЙ СТИЛЬ"
2. Нажать на кнопку "ЗАПИСАТЬСЯ НА ПРОБНОЕ ЗАНЯТИЕ"
3. В поле "Имя" ввести "Анна-Мария"
4. В поле "Телефон" ввести "7315640"
5. В поле "Пожелания" ввести "Хочу на пробное занятие по K-pop в субботу"
6. Нажать кнопку "Отправить"

**Ожидаемый результат:**
- Форма успешно отправляется
- Появляется сообщение об успешной отправке (например, "Спасибо! Мы свяжемся с вами")
- Модальное окно закрывается
- Данные передаются на сервер (или в консоль)

**Вид тестирования:** черный ящик, позитивное тестирование, функциональное тестирование, ручное тестирование

**Техника тест-дизайна:** Классы эквивалентности (валидные данные)
