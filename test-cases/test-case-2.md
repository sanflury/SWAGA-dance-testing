## Тест-кейс №2: Отправка пустой формы

**Описание:** Проверить поведение системы при попытке отправить форму без заполнения обязательных полей

**Предусловие:**
- Открыта главная страница сайта
- Открыто модальное окно записи (после клика на кнопку "ЗАПИСАТЬСЯ")

**Шаги:**
1. Оставить все поля пустыми
2. Нажать кнопку "Отправить"

**Ожидаемый результат:**
- Форма НЕ отправляется
- Поля "Имя" и "Телефон" подсвечиваются красным
- Появляются сообщения об ошибке: "Поле обязательно для заполнения"
- Модальное окно не закрывается
- Пользователь остается на форме для исправления

**Вид тестирования:** черный ящик, негативное тестирование, функциональное тестирование, ручное тестирование

**Техника тест-дизайна:** Классы эквивалентности(невалидные данные), анализ граничных значений (пустые значения)
