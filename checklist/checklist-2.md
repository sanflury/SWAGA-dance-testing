# 2. Функциональное тестирование формы записи

## 2.1 Поле "Имя"
- [ ] **Разрешенные символы:** можно ввести русские и английские буквы
- [ ] **Запрещенные символы:** цифры и спецсимволы НЕ вводятся (или показывается ошибка)
- [ ] **Двойные имена:** символ дефис (-) разрешен
- [ ] **Обязательность:** поле обязательно для заполнения
- [ ] **Граничные значения:**
    - Минимальная длина (2 символа)
    - Максимальная длина (30 символов)
    - Пробелы в начале/конце должны обрезаться или игнорироваться

## 2.2 Поле "Телефон"
- [ ] **Формат ввода:** только цифры (возможно + и пробелы/скобки для красоты)
- [ ] **Длина:** ровно 7 цифр (для коротких номеров) или 11-12 (для полных)
- [ ] **Маска ввода:** номер автоматически форматируется (+375 __ ___ __ __)
- [ ] **Обязательность:** поле обязательно для заполнения
- [ ] **Валидация:** проверка на недопустимые символы (буквы не проходят)

## 2.3 Поле "Пожелания/Комментарий"
- [ ] **Необязательность:** форму можно отправить без комментария
- [ ] **Максимальная длина:** проверить ограничение (например, 500 символов)
- [ ] **Многострочность:** текст переносится, можно вводить абзацы
- [ ] **Спецсимволы:** любые символы допустимы

## 2.4 Работа модального окна
- [ ] **Открытие:** окно появляется при клике на кнопку "Записаться"
- [ ] **Закрытие через крестик:** клик на крестик закрывает форму
- [ ] **Закрытие через оверлей:** клик вне формы (на затемненную область) закрывает окно
- [ ] **Escape:** нажатие клавиши Esc закрывает модальное окно
- [ ] **Возврат фокуса:** после закрытия фокус возвращается на кнопку открытия

## 2.5 Отправка формы
- [ ] **Успешная отправка:** при заполнении всех обязательных полей корректными данными форма отправляется
- [ ] **Сообщение об успехе:** после отправки появляется уведомление "Спасибо, мы свяжемся с вами"
- [ ] **Очистка формы:** после успешной отправки поля очищаются 
- [ ] **Защита от спама:** проверка на повторную отправку (кнопка блокируется на время)
