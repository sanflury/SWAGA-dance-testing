# 1. UI/UX тестирование

## 1.1 Навигация
- [ ] **Возврат на главную:** с любой внутренней страницы (хореографы, расписание) можно вернуться на главную за 1 клик:
    - Через логотип в шапке
    - Через пункт меню "Главная"
- [ ] **Работа меню:** все пункты меню ведут на соответствующие страницы/разделы
- [ ] **Якорные ссылки:** при клике на #стили, #faq, #контакты страница плавно скроллит к нужному разделу

## 1.2 Читаемость и типографика
- [ ] **Размер шрифта:** текст комфортно читать без увеличения (не менее 14px для основного текста)
- [ ] **Контрастность:** текст хорошо виден на фоне (проверить белый текст на видео/картинках)
- [ ] **Межстрочный интервал:** строчки не слипаются 
- [ ] **Выделение:** ключевые мысли выделены жирным шрифтом
- [ ] **Горизонтальный скролл:** текст не выходит за границы экрана

## 1.3 Адаптивность
- [ ] **Структура:** блоки не наезжают друг на друга при различной ширине экрана
- [ ] **Изображения:** картинки не обрезаются и не растягиваются
- [ ] **Кнопки:** достаточно крупные для нажатия пальцем
- [ ] **Тач-мишени:** между кликабельными элементами есть отступы (чтобы не промахиваться)

## 1.4 Интерактивность и обратная связь
- [ ] **Hover-эффекты:** при наведении на кнопки/ссылки меняется цвет/подсветка
- [ ] **Cursor:** при наведении на кликабельные элементы курсор меняется на pointer
- [ ] **Активное состояние:** при нажатии кнопка визуально меняется
- [ ] **Фокус:** при навигации клавиатурой видно, на каком элементе фокус
