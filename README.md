# Тестирование сайта SWAGA Dance Studio

## О проекте
Сайт танцевальной студии SWAGA. В этом репозитории собрана тестовая документация: чек-листы, тест-кейсы и баг-репорты

## Ссылки
- [Сам сайт](https://sanflury.github.io/SWAGA-dance/)
- [Ссылка на репозиторий с кодом](https://github.com/sanflury/SWAGA-dance)

## Структура
- `checklist/` — чек-листы для проверки функциональности
- `test-cases/` — подробные тест-кейсы
- `bug-reports/` — отчеты о найденных дефектах
