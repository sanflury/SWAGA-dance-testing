# Баг-репорт: BUG-001

**Заголовок:** Отсутствует ссылка на Instagram в футере сайта

**Проект:** SWAGA Dance Studio
**Компонент:** Футер / Социальные сети
**Окружение:** Windows 10, Firefox 148.0.2

**Серьезность:** Minor
**Приоритет:** High

**Шаги воспроизведения:**
1. Открыть главную страницу сайта
2. Прокрутить вниз до футера
3. Нажать на иконку Instagram

**Фактический результат:**
- Визуально кнопка реагирует на нажатие (меняет цвет)
- Дальше ничего не происходит
- В консоли браузера ошибок нет
- Ссылка не открывается

**Ожидаемый результат:**
При нажатии на иконку Instagram должна открываться страница студии в Instagram в новой вкладке
